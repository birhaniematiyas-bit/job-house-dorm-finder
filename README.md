# Job, House, and Dorm Finder

A simple web-based platform designed to help users in Ethiopia find **jobs**, **houses for rent**, **houses for sale**, **workers**, and **dorm/room rentals**.

This project aims to connect people easily by providing a single place to search and post listings.
